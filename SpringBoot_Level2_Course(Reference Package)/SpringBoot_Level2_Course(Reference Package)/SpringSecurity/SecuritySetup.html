Setting up Spring Security Defaults

Adding Spring Security Dependency

<dependency> <groupId>org.springframework.boot</groupId> <artifactId>spring-boot-starter-security</artifactId> </dependency>

Adding this dependency automatically:

  • Secures all endpoints
  • Generates a default password (logged in console)
  • Creates a default user "user"
  • Enables basic authentication

Default Security Configuration

Spring Boot provides sensible security defaults that can be customized as needed.

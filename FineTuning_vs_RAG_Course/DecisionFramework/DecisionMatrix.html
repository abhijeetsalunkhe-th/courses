The Decision Matrix

Decision Flowchart

Ask these questions in order to choose the right approach:

Question 1: Does the model already know enough?

  • Yes: Use Prompt Engineering. Start here always.
  • No: Continue to Question 2.

Question 2: Does the knowledge change frequently?

  • Yes: Use RAG. Documents can be updated without retraining.
  • No: Continue to Question 3.

Question 3: Do you need behavioral change or knowledge?

  • Knowledge: Use RAG. More reliable for factual grounding.
  • Behavior/Style: Use Fine-Tuning. Better for tone, format, domain language.

Hybrid Approaches

In practice, the best solutions often combine approaches: Fine-tuned model + RAG + careful prompting. Use fine-tuning for domain language and style, RAG for current knowledge, and prompt engineering for task-specific instructions.

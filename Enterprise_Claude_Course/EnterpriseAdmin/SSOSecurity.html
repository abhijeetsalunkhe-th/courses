SSO, Security, and Compliance

Single Sign-On (SSO) Integration

Enterprise Claude supports SAML 2.0 SSO for centralized authentication:

  • Supported providers: Okta, Azure AD, Google Workspace, OneLogin, PingFederate
  • SCIM provisioning: Automatic user provisioning and de-provisioning
  • Just-in-time (JIT) provisioning: Create accounts on first SSO login
  • Enforced SSO: Require all users to authenticate via identity provider

Compliance and Certifications

Anthropic maintains enterprise-grade compliance certifications:

  • SOC 2 Type II: Security, availability, and confidentiality controls
  • HIPAA: Available for healthcare use cases with BAA
  • GDPR: Data processing agreements and EU data residency options
  • CSA STAR: Cloud security assessment

Usage Controls and Budgets

Prevent unexpected costs with built-in controls:

  • Set monthly spending limits per workspace
  • Configure per-key rate limits (requests/minute, tokens/day)
  • Set up budget alerts at 50%, 80%, and 100% thresholds
  • Auto-disable keys when budgets are exceeded

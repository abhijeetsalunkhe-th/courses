Gemini in Docs, Slides, and Gmail

Gemini in Google Docs

Gemini integrates directly into Google Docs for writing assistance:

  • Help me write: Generate drafts from prompts or bullet points
  • Summarize: Create summaries of long documents
  • Rewrite: Change tone (formal, casual, concise), elaborate, or shorten text
  • Proofread: Fix grammar, spelling, and style issues

Effective Prompt Patterns for Docs

TaskPrompt Pattern
Draft an email"Write a professional email to [recipient] about [topic], tone: [formal/friendly]"
Create a proposal"Draft a project proposal for [project] including objectives, timeline, and budget"
Meeting notes"Convert these bullet points into formatted meeting notes with action items"
Job description"Write a job description for [role] at [company type] requiring [skills]"

Gemini in Gmail

  • Draft replies: Generate contextual reply suggestions based on email thread
  • Summarize threads: Get a quick summary of long email chains
  • Adjust tone: Make responses more formal, casual, or concise
  • Smart compose: AI-powered autocomplete while typing

Gemini in Google Slides

  • Generate slides: Create presentation outlines and content from a topic
  • Image generation: Create custom images for slides using text descriptions
  • Speaker notes: Generate speaker notes based on slide content
  • Formatting: Suggest layout improvements and visual consistency

What Is Extended Thinking?

Overview

Extended thinking gives Claude a dedicated "thinking" phase before generating its response. During this phase, Claude can reason through complex problems, explore multiple approaches, and arrive at more accurate answers.

How It Works

  • Thinking tokens: Claude uses internal reasoning tokens (visible to you) before the final response
  • Budget control: You set a max_tokens budget for thinking
  • Transparent reasoning: You can see Claude's thought process in thinking content blocks

API Usage

response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=16000,
    thinking={
        "type": "enabled",
        "budget_tokens": 10000  # Up to 10K tokens for thinking
    },
    messages=[{
        "role": "user",
        "content": "Analyze the time complexity of this recursive algorithm..."
    }]
)

# Access thinking and response separately
for block in response.content:
    if block.type == "thinking":
        print("THINKING:", block.thinking)
    elif block.type == "text":
        print("RESPONSE:", block.text)
